## Run project

In order to run project you need to follow next steps:
- Clone repository.
- Navigate in project's folder.
- Run composer install.
- Copy .env.example to .env (cp .env.example .env) .
- Generate application key: php aritsan key:generate .
- Run project: php aritsan serve .
- Enjoy ;))
